# 4주차 실습 정리

## 오늘 한 것
- 원형 / AABB / OBB Bounding Box 시각화
- 세 방식의 충돌 판정 차이 관찰
## 관찰 내용
AABB와 OBB그리고 CIRCLE이 오브젝트에 어떻게 적용되는지 관찰했다.
### 원형 vs AABB
(직접 관찰한 내용 작성)
원형은 충돌거리만 계산해서 일정했지만 사각형에 적용하면 원이 오브젝트보다 커지거나 작아져 빈공간임에도 충돌하거나 오브젝트끼리 접촉해도 HIT가 안떴다. AABB는 사각형에 알맞는것같다  
### AABB vs OBB
(직접 관찰한 내용 작성)
AABB는 오브젝트가 회전하면 실제 모양보다 충돌 범위가 더 크게 잡혀 부정확했고 OBB는 물체의 회전을 반영해서 실제 형태에 맞게 충돌을 판정할 수 있어 더 정확했다.
## AI와의 대화에서 배운 것
(흥미로웠던 AI 답변 요약)
원형 충돌 감지에서 math.sqrt를 꼭 사용하지 않아도 된다는 점이 가장 흥미로웠다.
## 내 게임에 적용한다면
(어떤 방식을 쓸 것인지, 이유는?)
내 게임에는 AABB를 쓸것같다 이유는 AABB는 구현이 간단하고 계산 속도가 매우 빨라서, 실시간으로 계속 충돌을 검사해야 하는 게임에 적합하기 때문이다.
