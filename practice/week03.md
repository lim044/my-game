# week 3 실습 기록

## 목표
플레이어 캐릭터를 키보드로 움직이기 구현

## AI 대화 기록

**Q1:FPS를 화면에 표시하려면?**

- AI 답변: clock.get_fps()와 pygame.font로 텍스트 출력
- 시행착오: font 초기화 안 해서 에러 / FPS 값이 계속 0으로 보임
- 적용 결과: 매 프레임마다 get_fps() 값 렌더링해서 정상 출력 완료

**Q2: WASD로 원을 움직이려면?**

- AI 답변: pygame.key.get_pressed()로 키 입력을 계속 확인해서 좌표 변경
- 시행착오: 시행착오: KEYDOWN만 사용해서 한 번만 움직임
- 최종 해결: get_pressed() + speed 변수로 자연스러운 이동 구현 완료

**Q2: 화면 밖으로 원이 못 나가게 하려면?**

- AI 답변: 좌표를 화면 크기 안으로 제한 (clamp)
- 시행착오: 0~800, 0~600으로만 제한해서 원이 반쯤 잘림
- 최종 해결: 반지름을 고려해서 max(radius, min(x, width - radius)) 적용 완료
